# 用户手册

## 1. 系统简介

学校部门财务管理系统用于管理校内部门的收入、支出、账户、类别、统计报表和数据备份。系统提供管理员和财务员两类角色，适合课程设计演示与基础业务处理。

## 2. 运行说明

### 2.1 前端运行

1. 进入 `10-源代码/frontend`
2. 执行 `npm install`
3. 执行 `npm run dev`

### 2.2 后端运行

1. 进入 `10-源代码/backend`
2. 配置 MySQL 数据库连接
3. 导入 `11-数据库/财务管理系统.sql`
4. 执行 `mvn spring-boot:run`

## 3. 登录说明

- 管理员和财务员均使用用户名密码登录。
- 登录成功后进入首页仪表盘。
- 页面右上角可退出系统。

## 4. 功能操作说明

### 4.1 用户管理

适用角色：管理员

- 进入“用户管理”页面。
- 点击“新增用户”。
- 输入用户名、姓名、电话和角色。
- 保存后用户可登录系统。

### 4.2 部门管理

适用角色：管理员

- 进入“部门管理”页面。
- 可新增、修改、停用部门。
- 已被业务引用的部门建议停用而非删除。

### 4.3 账户管理

适用角色：管理员

- 进入“账户管理”页面。
- 维护账户编号、名称、所属单位和说明。

### 4.4 类别管理

适用角色：管理员

- 收入类别和支出类别分别维护。
- 若类别已被业务引用，系统应禁止直接删除。

### 4.5 收入录入

适用角色：管理员、财务员

1. 进入“收入记录”页面。
2. 点击“新增收入”。
3. 选择部门、账户、收入类别。
4. 输入金额、日期、经办人和备注。
5. 保存后可在列表中查询。

### 4.6 支出录入

与收入录入操作类似，类别改为支出类别。

### 4.7 报表查询

适用角色：管理员、财务员

1. 进入“报表中心”页面。
2. 选择周报、月报、年报或自定义区间。
3. 可选指定部门。
4. 点击查询后查看汇总值、分类占比和趋势结果。

### 4.8 备份管理

适用角色：管理员

1. 进入“备份管理”页面。
2. 点击“创建备份”。
3. 查看备份记录列表。
4. 需要恢复时点击“恢复”，并进行二次确认。

## 5. 常见问题

### 5.1 无法登录

- 检查用户名和密码是否正确
- 检查账号是否被停用
- 检查后端服务是否正常启动

### 5.2 页面没有数据

- 检查数据库是否已导入初始化脚本
- 检查查询条件是否过窄
- 检查当前角色是否有权限

### 5.3 不能删除主数据

若部门、账户或类别已被业务记录引用，系统按设计只允许停用，不允许物理删除。

## 6. 注意事项

- 登录后请根据角色使用对应功能。
- 备份与恢复操作仅限管理员。
- 数据恢复前应确认当前数据是否需要保留。
