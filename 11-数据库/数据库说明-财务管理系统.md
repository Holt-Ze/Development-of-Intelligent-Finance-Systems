# 数据库说明

## 1. 数据库名称

建议数据库名称为 `finance_management_system`。

## 2. 脚本内容

`财务管理系统.sql` 包含以下内容：

- 数据库创建语句
- 10 张核心业务表
- 角色和用户初始化数据
- 部门、账户、类别示例数据
- 收入和支出示例数据

## 3. 导入方法

1. 启动 MySQL 8
2. 创建数据库或直接执行脚本
3. 在 Navicat 或 MySQL 客户端中导入 `财务管理系统.sql`

## 4. 说明

- 密码字段为课程演示用摘要示例，不代表正式生产安全方案。
- 备份文件路径采用示例目录，需要根据运行环境调整。
