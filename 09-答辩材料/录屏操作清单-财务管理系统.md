# 财务管理系统录屏操作清单

## 1. 录屏前检查

- [ ] 确认 MySQL 已启动
- [ ] 确认已导入 `11-数据库/财务管理系统.sql`
- [ ] 确认后端数据库连接配置正确
- [ ] 确认后端可以正常启动
- [ ] 确认前端可以正常启动
- [ ] 确认浏览器已打开系统首页
- [ ] 确认录屏软件已开启麦克风
- [ ] 确认桌面无无关窗口和弹窗

## 2. 启动顺序

- [ ] 进入 `10-源代码/backend`
- [ ] 执行 `mvn spring-boot:run`
- [ ] 确认后端启动成功，端口为 `8080`
- [ ] 进入 `10-源代码/frontend`
- [ ] 如首次运行，执行 `npm install`
- [ ] 执行 `npm run dev`
- [ ] 打开前端本地地址

## 3. 演示账号准备

- [ ] 管理员账号：`admin / 123456`
- [ ] 财务员账号：`finance01 / 123456`

## 4. 正式录屏流程

### 4.1 开场

- [ ] 简短介绍项目名称
- [ ] 说明项目用途：学校部门财务管理
- [ ] 说明技术栈：Vue 3 + Spring Boot 3 + MySQL 8
- [ ] 说明本次演示内容：登录、基础数据、收支、报表、备份

### 4.2 运行环境展示

### 4.5 切换财务员

## 5. 录屏注意事项

- [ ] 录制前先自己完整走一遍流程
- [ ] 新增数据时尽量使用容易辨认的演示内容
- [ ] 报表演示前确认新增数据已经进入统计
- [ ] 恢复功能如果担心影响数据，可只演示确认弹窗
- [ ] 鼠标移动速度不要太快
- [ ] 每切换一个模块先停顿 1 到 2 秒
- [ ] 讲解尽量围绕“做什么、为什么、结果是什么”

## 6. 建议时长分配

- [ ] 开场与环境：1 分钟
- [ ] 管理员基础数据演示：1.5 分钟
- [ ] 财务员收支录入：2 分钟
- [ ] 报表统计：1.5 分钟
- [ ] 备份管理与总结：1 到 2 分钟

## 7. 结束前最终确认

- [ ] 录屏文件已保存
- [ ] 声音清晰可辨
- [ ] 画面分辨率正常
- [ ] 无明显卡顿或口误
- [ ] 如有问题，优先重录业务演示段落
