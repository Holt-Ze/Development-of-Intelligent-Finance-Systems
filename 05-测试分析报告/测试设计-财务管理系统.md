## 3. 测试项分解

### 3.1 登录与权限

| 测试点 | 方法 | 关注内容 |
| --- | --- | --- |
| 正确账号密码登录 | 等价类 | 返回令牌、用户名和角色集合 |
| 错误密码登录 | 反向测试 | 给出明确错误提示 |
| 停用账号登录 | 状态测试 | 禁止进入系统 |
| 页面权限控制 | 权限矩阵 | 财务员不可进入用户管理和备份管理 |

### 3.2 主数据管理

| 测试点 | 方法 | 关注内容 |
| --- | --- | --- |
| 新增主数据 | 正常场景 | 字段保存成功、列表可见 |
| 修改主数据 | 正常场景 | 修改内容生效 |
| 停用主数据 | 状态测试 | 停用后不再允许新增业务引用 |
| 重复编码或关键字段为空 | 边界/异常 | 表单提示或后端校验生效 |

### 3.3 收入与支出记录

| 测试点 | 方法 | 关注内容 |
| --- | --- | --- |
| 正常录入 | 场景法 | 自动生成编号、保存成功 |
| 金额为 0 或负数 | 边界值 | 拒绝保存 |
| 发生日期缺失 | 必填校验 | 阻止提交 |
| 引用停用主数据 | 状态测试 | 后端校验拦截 |
| 记录修改与状态切换 | 回归测试 | 修改成功且报表同步更新 |

### 3.4 报表统计

| 周报、月报、年报 | 等价类 | 时间粒度切换正确 |
| 自定义区间 | 边界值 | 起止日期口径正确 |
| 部门筛选 | 组合测试 | 指定部门与全量部门结果可区分 |

### 3.5 备份管理

| 测试点 | 方法 | 关注内容 |
| --- | --- | --- |
| 创建备份 | 正常场景 | 生成备份记录和文件信息 |
| 下载备份 | 场景法 | 文件可获取 |
| 恢复备份 | 关键流程 | 恢复动作受管理员权限控制 |
| 非管理员访问备份页 | 权限矩阵 | 前端和后端均拒绝 |

## 4. 测试方法设计

### 4.1 等价类设计

- 登录：正确账号密码、错误密码、空账号、空密码、停用账号。
- 金额字段：合法正数、零、负数、超大值、空值。
- 日期字段：合法日期、空值、起止反转。
- 角色：管理员、财务员、未登录用户。

### 4.2 边界值设计

- 金额边界：`0`、`0.01`、大额示例值。
- 日期边界：月初、月末、跨月、跨年。
- 文本字段边界：备注为空、常规长度、较长描述。

### 4.3 场景法设计

1. 管理员首次登录并完成主数据初始化。
2. 财务员录入收入和支出后查看报表变化。
3. 管理员创建备份并执行恢复验证。
4. 权限不足用户尝试访问受限页面并被拦截。

| --- | --- |
| 高 | 登录、权限、收支录入、报表统计、备份恢复 |
