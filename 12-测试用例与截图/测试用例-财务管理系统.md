# 测试用例

## 1. 功能测试用例

| 编号 | 功能模块 | 测试步骤 | 预期结果 |
| --- | --- | --- | --- |
| TC-01 | 登录 | 输入正确用户名和密码 | 登录成功并进入首页 |
| TC-02 | 登录 | 输入错误密码 | 提示用户名或密码错误 |
| TC-03 | 登录 | 使用停用账号登录 | 提示账号已停用 |
| TC-04 | 用户管理 | 管理员新增财务员 | 新增成功 |
| TC-05 | 部门管理 | 新增部门信息 | 保存成功并显示在列表 |
| TC-06 | 账户管理 | 新增账户信息 | 保存成功 |
| TC-07 | 收入类别 | 新增收入类别 | 保存成功 |
| TC-08 | 支出类别 | 新增支出类别 | 保存成功 |
| TC-09 | 收入录入 | 提交合法收入记录 | 保存成功 |
| TC-10 | 支出录入 | 提交合法支出记录 | 保存成功 |
| TC-11 | 收入录入 | 提交负数金额 | 校验失败 |
| TC-12 | 报表查询 | 查询月报表 | 返回收入、支出、结余和统计明细 |
| TC-13 | 报表查询 | 按部门查询自定义区间报表 | 返回该部门统计结果 |
| TC-14 | 主数据引用 | 删除已被引用的类别 | 系统禁止删除 |
| TC-15 | 备份 | 管理员创建备份 | 生成备份记录 |
| TC-16 | 备份 | 财务员访问备份功能 | 提示无权限 |
